How is it done ?

To determine the favorites for an edition, we have built a profile for each and every single team in the world. This profile is based uppon several criterias such the number of wins, the number of goals scored or the FIFA rank for example.
These profiles are then compared to the of the typical winner which is defined as the average profile of every single historic winner. The favorites for an edition are the teams with the closer profile to the winner profile.
